Versão inicial:
Um sistema de batalha inspirado nos jogos Pokémon
É possível definir o nível, o nome e o tipo do Pokémon do usuário e do rival, visualizar seus status e realizar batalhas de forma automatizada. Os Pokémon se atacam sem a necessidade de comandos do usuário, e vence aquele cujo elemento tem vantagem sobre o do adversário.
